# Plano de estudos: Logica com Java para processos seletivos

Este plano foi feito para estudo pratico de logica em Java (sem Spring), com foco em desafios de entrevistas tecnicas.

## Como usar

- Duracao: 8 semanas
- Ritmo sugerido: 5 dias por semana, 1h30 a 2h por dia
- Formato diario:
  - 15 min: revisao curta
  - 60-90 min: desafios cronometrados
  - 20 min: correcao e refatoracao
  - 10 min: registro de erros

## Regras de execucao

- Sempre escrever uma ideia de solucao antes de codar.
- Sempre anotar complexidade esperada (tempo e memoria).
- Sempre testar 3 cenarios: normal, borda e grande.
- Se travar por 20 min, ler dica e repetir o problema no dia seguinte sem consulta.

## Controle de evolucao (semanal)

Registre no fim de cada semana:

- Taxa de acerto (%).
- Tempo medio por questao easy e medium.
- Quantidade de erros por categoria:
  - sintaxe
  - logica
  - caso de borda
  - complexidade
  - gestao de tempo
